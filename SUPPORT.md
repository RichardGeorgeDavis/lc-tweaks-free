# Support

For general questions, start with GitHub Discussions if available:

https://github.com/RichardGeorgeDavis/lc-tweaks-free/discussions

For reproducible bugs, open a GitHub issue:

https://github.com/RichardGeorgeDavis/lc-tweaks-free/issues

For private support or commercial enquiries, contact Lucidity Design:

https://lucidity.design/contact/

## Helpful Support Details

When asking for help, include:

- WordPress version.
- PHP version.
- LC Tweaks version.
- Active theme and whether Divi is active.
- WooCommerce version if the issue is WooCommerce-related.
- The option or screen involved.
- Error logs or screenshots where relevant.
