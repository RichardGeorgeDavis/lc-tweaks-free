# Security Policy

## Supported Versions

Security fixes are handled for the current public release line of LC Tweaks.

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Report security concerns directly through Lucidity Design:

https://lucidity.design/contact/

Include:

- A clear description of the issue.
- Steps to reproduce.
- Affected versions if known.
- Any logs, screenshots, or proof-of-concept details that help verify the issue.

Security reports will be reviewed before public disclosure or release notes are published.
