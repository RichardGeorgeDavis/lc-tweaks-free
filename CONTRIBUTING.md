# Contributing

Thanks for helping improve LC Tweaks.

## Before Opening an Issue

- Check the latest release.
- Confirm whether the behavior happens with only LC Tweaks and the relevant dependency active.
- Note whether the site uses Divi, WooCommerce, both, or neither.
- Include the WordPress, PHP, theme, Divi, WooCommerce, and LC Tweaks versions where relevant.

## Good Bug Reports Include

- What you expected to happen.
- What actually happened.
- Steps to reproduce.
- Screenshots or short screen recordings when the issue is visual.
- Any relevant PHP error log entries.

## Feature Requests

Feature requests are welcome when they describe a real site workflow or recurring problem. Please include:

- The problem you are solving.
- How you currently work around it.
- Whether the feature should apply to WordPress generally, Divi, WooCommerce, or maintenance tooling.

## Pull Requests

This public repository is generated from a private source repository. Direct pull requests may not be merged as-is. If you want to propose code, open an issue first so the change can be reviewed and applied through the source workflow.

## Development Notes

LC Tweaks Free is not Divi-dependent. Divi-specific options appear only when Divi is active, and WooCommerce-specific options appear only when WooCommerce is active.
